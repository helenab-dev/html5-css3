HTML - HyperText Markup Language

HTML é focada em contúdo(textos, imagens, vídeos, tabelas)

style é CSS e não html.
<mark style="background-color: rgb(242, 0, 255);">um texto marcado</mark> - esse jeito precisaria fazer para cada mark
caso eu queira mudar todos de uma vez(dessa página), eu vou no cabeçalho e utilizo a tag style e mark:
<style>
        mark {
            background-color: aquamarine;
        }
    </style>
